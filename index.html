MoveIt File Utility

Python

A script with a graphical user interface that allows user to easily transfer files from one directory to up to two different directories (primary and backup destinations).

MoveIt can sort files into sub-directories based on the files' dates or into a subdirectory of the user's choice. Conceived in order to automate transferring photos from an SD card to two hard drives, while sorting the photos into different subdirectories based on their creation dates.
